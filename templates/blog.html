Reconocimiento Facial

Este proyecto consiste de tres módulos:

1. Web Scrapping.

2. Red Neuronal Convolucional

3. Cámara de Python

En el primer módulo, se hizo scraping de Google Images para crear el dataset con el que se entrenaría la red neuronal. Para ello se utilizó Selenium, para obtener las URLs de las imágenes. Después con Requests se descargaban y con Pillow se guardaban en una estructura de directorios. Después de esto, se utilizó cv2 y Pillow para limpiar y recortar las imágenes obtenidas. Con las imágenes pre procesadas, se creó un modelo secuencial con Keras (red neuronal convolucional). Para alimentar el modelo, se utilizó un generador de datos de imagen. Finalmente se utilizó cv2 para implementar un proceso en el que se toma una foto del usuario, se pasa la foto al modelo, y se hace la predicción.


¿Cómo funciona?


Con Selenium, se hace una búsqueda en Google por cada celebridad que queremos identificar y se obtienen las URL’s de los resultados. Después se hacen llamadas a estas URL’s para descargar las imágenes y guardarlas en el formato requerido con Pillow. Con Keras, se crea una Red Neuronal en el que el input son arrays de tres dimensiones (dos espaciales y una de color). Y el output es una etiqueta con la predicción. Las clases y el entrenamiento se hacen pasando imágenes de un generador que leía la estructura de los directorios en los que se guardaron las imágenes. El modelo es una red Neuronal Convolucional. Esto significa que entre la capa de input y la de output hay capas ocultas, en las que se procesan las imágenes mediante convoluciones, transformaciones espaciales y operaciones aritméticas). Este modelo se evalúa con base en la precisión con la que logra clasificar las imágenes. Finalmente, se toma una imagen del usuario de la página, la cual se alimenta al modelo para tener la predicción definitiva.
