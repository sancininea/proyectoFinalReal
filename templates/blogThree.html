Tax Dashboard

Dashboard Financiero


Este dashboard utiliza la mayoría de las herramientas aprendidas durante este diplomado: Python, PostgreSQL, FLask, Jinja, JQuery, Bootstrap y HTML.



Es un sistema de control fiscal basado en archivos de Comprobantes Fiscales Digitales mejor conocidos como Facturas electrónicas o CFDI. Estos se encuentran en formato XML. Cada uno de los documentos representa una compra, venta, pago o cobro de servicios. El sistema puede leer los archivos de CFDI en XML directamente de un folder en el equipo o desde el webservice de consulta que proporciona el Servicio de Administración Tributaria. Los archivos XML que se extraen del webservice proporcionado por el SAT, utilizan la Firma Electrónica de una empresa o persona física.

Para realizar la conexión y extracción usamos una biblioteca de Python que crea el cliente de servicios a partir de una URL, obtiene un “token” de seguridad para conectarse, solicita el servicio y posteriormente monitorea el estatus de la petición para poder descargar una serie de archivos codificados (base 64) que al unirse forman un archivo ZIP con la información solicitada. Estos archivos se leen e ingresan a la base de datos para nutrir las gráficas y cuadros del dashboard. También se realiza un webscrapping para obtener los listados de empresas con irregularidades fiscales (Lista negra del SAT conforme a los Artículos 69 y 69-B del Código Fiscal de la Federación) y se integran los datos a la base para poder cruzarse frente a los CFDI.
